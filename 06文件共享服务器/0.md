## 文件共享服务器

### 共享服务器概述

通过网络提供文件共享服务，提供文件下载和上传服务（类似于FTP服务器）

### 创建共享

`属性` - 	`共享`  - `开启共享` --- `设置共享名` -- `设置共享权限`

注意：

1. 在本地登录时 ，只收到NTFS权限的影响
2. 在远程登录时，将受到共享及NTFS权限的共同影响，且取`交集`
3. 所以建议共享权限设为everyone完全控制，然后具体权限根据NTFS权限设置即可

### 访问共享

```
在电脑地址栏上，输入NUC地址
\\文件共享服务器IP
\\文件共享服务器IP\共享名（不一定是文件名）
```

### 查看共享

```
net share		# 显示所有共享
net share 共享名 /del	# 删除共享文件
```

### 屏蔽系统共享文件产生

#### 打开注册表

`regedit`

#### 定位共享注册表位置

```
HKEY_Local_MACHINE\System\CurrentControlSet\Services\LanmanServer\Parameters\
右键新建 REG_DWORD类型的AutoShareServer 键，值为0
```

### 查看本地网络连接状态

```
netstat -an
```

### 关闭445服务

445服务就是共享服务端口

```
方法1： 打开 services.msc, 禁用server 服务
方法2：禁止被访问445，配置高级防火墙入站规则
```

### 重新登录文件共享服务器

使用命令 `net use`可以看到最近的网络连接，使用命令

```
net use \\10.1.1.1\共享文件名 /del 
```

就可以删除对应的网络连接记录，就可以重新登陆了。
